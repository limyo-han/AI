***** 3. AI란? 데이터의 특성을 잘 분석하는 알고리즘.

C언어 등) Data -> Rule -> Result
Data, Rule을 입력하고 Result를 얻는다.
예시. 데이터 -> if else -> 분류 결과

AI) Data -> Result(Label) -> Rule
Data, Result(Label)를 입력하고 Rule을 얻는다. (new input을 넣으면 output이 나온다. 여기는 test, 앞은 train)
Label : AI에서는 데이터가 어떤 클래스(결과)에 속하는지 알려주는 값

훈련 데이터(fit), 테스트 데이터(predict). 만든 사람들은 Rule을 알 필요가 없다(?)

데이터 입력 방법 3가지
1. 파일로부터 받기.
2. 라이브러리부터 입력받기.
3. 웹 주소로부터 입력받기.

데이터를 받으면 먼저 x(Data)와 y(Result)를 나눈다.
그리고 training을 위한 데이터와 test를 위한 데이터를 나눈다. 비율은 보통 7:3 
training과 test를 하고 predict된 값과 실제 값을 비교한다(정확도 검사).

3. Train, Test 왜 구분하는가?  

AI 안에 머신러닝, 그 안에 딥러닝.
머신러닝은 특징을 추출하고 모델에 집어넣는다. 특징에 있어서 전문가의 지식이 필요함.
딥러닝은 로우(raw)데이터를 집어넣는다. 그러면 알아서 특징을 추출한다. 머신러닝에서 특징을 추출해서 딥러닝에 넣으면 효과가 좋다.
