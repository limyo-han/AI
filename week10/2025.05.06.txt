이미지를 위한 신경망
-> 컨벌루션 신경망

dense layer
-> 층을 다 더하는 것
각 특성에 가중치(w) 곱하고 절편 더해서 1개의 출력을 내는 것


CNN(Convolution Neural Network)
->하위 레이어의 노드들과 상위 레이어의 노드들이 부분적으로만 연결

합성곱
특성 부분부분을 그룹으로 묶는 것, 출력이 많아지지만 특성 간의 연관성 추출 가능


FCN(dense) layer
= N의 입력을 하나로 처리
CNN
1. N개의 입력을 필터(소규모그룹)단위로 여러개 추출
2. 필터를 사용하여 여러 개의 특징 추출


- convolution 연산이 무엇인가?
두 개의 함수(f=원래 이미지, g=필터)를 새로운 값을 만들어낸다.
다양한 특징을 추출하기 위한 연산


- 필터란 무엇인가?
"작은 가중치 행렬", 이미지의 특징을 추출하는데 사용되는 도구(예, 3*3, 5*5)

합성곱
작은 필터를 뽑을 뿐만 아니라, 원본하고 필터를 곱해서 여러가지 특징을 뽑아내는 것

stride의 역할?
필터를 얼만큼 이동할 것인가(작을수록 연산 up)

필터의 가중치는 학습되면서 적용

풀링(Pooling) -> 입력 데이터의 크기를 줄이는 것, 이동성에 불편
convolution 연산으로 인한 정보 급증가를 위함

CNN과 풀링을 반복하여(다층 신경망) 결과 도출

2차원 -> dense 평탄화(일렬 -> flatten layer -> 출력)
